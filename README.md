# Laftel Plus

라프텔(laftel.net)에서 애니 시청을 편하게 해주는 크롬 확장 프로그램.

- **오프닝 스킵** — 라프텔의 "오프닝 스킵" 버튼이 뜨면 자동 클릭
- **엔딩 스킵** — "엔딩 스킵" 버튼이 뜨면 자동 클릭
- **정주행 모드** — 영상이 끝나면 "다음 화" 버튼을 자동 클릭

팝업은 NordVPN 스타일로, 가운데 큰 전원 버튼과 아래 토글 3개로 구성됩니다.
세 토글 중 **하나라도 켜지면 활성화**, **전부 꺼지면 확장이 꺼진 상태**가 되고
전원 버튼이 이 상태를 표시합니다. 전원 버튼을 누르면 활성 상태일 땐 전부 끄고
(직전 조합을 기억), 꺼진 상태일 땐 그 조합을 복원합니다. 설정은
`chrome.storage.sync`로 기기 간 동기화됩니다.

## 동작 방식

라프텔은 React 기반이라 CSS 클래스명이 빌드마다 바뀝니다. 그래서 이 확장은
버튼의 **텍스트 / aria-label**("스킵", "건너뛰기", "다음 화" 등)을 기준으로
버튼을 감지합니다. MutationObserver + 2초 간격 백업 스캔으로 버튼 등장을
감지하고, 카테고리별 쿨다운으로 중복 클릭을 방지합니다.

"스킵"이라고만 표시된 버튼은 현재 재생 위치(전체의 60% 기준)로
오프닝/엔딩을 구분합니다.

수집하는 데이터 없음. 권한은 `storage`(설정 저장)와 laftel.net 콘텐츠
스크립트뿐입니다.

## 개발용 설치 (테스트)

1. `chrome://extensions` 접속
2. 우측 상단 **개발자 모드** 켜기
3. **압축해제된 확장 프로그램을 로드합니다** → 이 폴더 선택
4. laftel.net에서 애니 재생 후 스킵/다음화 동작 확인

코드를 수정하면 `chrome://extensions`에서 새로고침(↻) 필요.

## Chrome Web Store 제출

1. `./package.sh` 실행 → `laftel-plus-vX.X.X.zip` 생성
2. [Chrome Web Store 개발자 대시보드](https://chrome.google.com/webstore/devconsole)
   접속 — 최초 1회 등록비 $5 필요
3. **새 항목** → zip 업로드
4. 스토어 등록정보 작성:
   - 설명, 카테고리(예: 엔터테인먼트), 언어(한국어)
   - 스크린샷 최소 1장 (1280×800 또는 640×400) — 라프텔 재생 화면에서
     팝업을 연 모습을 캡처하면 좋음
   - 아이콘 128×128은 `icons/icon128.png` 사용
5. **개인정보 보호 관행** 탭:
   - 각 권한(`storage`, 호스트 접근)의 사용 목적 한 줄씩 작성
   - "데이터를 수집하지 않음" 체크
   - 개인정보처리방침 URL은 데이터를 수집하지 않으면 필수는 아니지만,
     GitHub에 간단한 문서를 올려 링크하면 심사가 수월함
6. 제출 → 심사는 보통 1~3일

### 주의: 상표 관련

확장 이름에 "Laftel"이 들어가면 상표 정책으로 반려될 수 있습니다.
반려되면 이름을 "오토스킵 for 라프텔"처럼 `for` 형태로 바꾸고, 설명에
"라프텔 공식 확장이 아닙니다" 문구를 추가하세요.

### 주의: 사이트 업데이트

라프텔이 버튼 문구를 바꾸면 감지가 실패할 수 있습니다. 그 경우
`content.js` 상단의 정규식(`RE_SKIP`, `RE_NEXT` 등)만 고치면 됩니다.
